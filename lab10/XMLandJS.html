About XML and JS

1. What is XML?

XML stands for Extensible Markup Language just like html and it was developed by the W3C as a flexible method to create common information formats and share the format and the information on the Web and carry data.

2. How is XML used?

It provides web publishing in which it constructs interactive pages and customers may customize as well . Additionally, you can save the information once and then render it for many viewers. It is used for transmitting data across systems, frequently between systems from different corporations, using syntax with slashes and brackets. It creates formats for data that is used to encode information for documentation, database records, transactions and many other types of data. It enables data sharing across businesses without mandating that their internal files follow a specific format. It's ised for web, online business, and mobile apps and other applications. As a result, it is simpler for devices and applications of all kinds to utilize, store, send, and display data.

What is JavaScript?

JavaScript is a programming language used for creating and manage dynamic and interactive website content such as anything that moves, refreshes, or otherwise changes on your screen without necessitating a manual page reload.

How is JavaScript related to HTML and CSS?

The basic structure of websites is provided by HTML, which is subsequently enhanced and altered by additional programs like CSS and JavaScript. Presentation, formatting, and layout are controlled via CSS. JavaScript is used to regulate how certain elements behave. JavaScript is added to an HTML web page using script tag, and similar to CSS, code can either be written inside the tag or in it's own seperate file.

What are some common JavaScript elements, properties, methods, and functions?

Elements

  • Document Object Model(DOM): Hierarchial structure can be used to access page elements and apply styles
  • Window Object: Window object methods refer to the functions created inside the Window Object, which can be used to perform various actions on the browser window

Properties

  • clientLeft: This is a property which returns the width of the element's left border, in pixels.
  • childElementCount: This is a property which returns the number of child elements of an element.

Methods

  • clearInterval(): method clears a timer set with the setInterval() method
  • The blur(): method removes focus from a window.

Functions

  • parseInt: parses a value as a string and returns the first integer.
  • parseFloat(): parses a value as a string and returns the first number
  • isFinite(): returns true if a value is a finite number.
